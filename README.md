# Full-Stack-Web-Workout-Tracker
Full-Stack Exercise Collection Web App

To use this web app, you must first install Node.js on your laptop. This can be done here: https://nodejs.org/en/download

**After installing Node.js:**
Open two separate terminals within your IDE.
Run "npm run dev" in the directory containing the arnettj_react file. 
Run "npm start" in the directory containing the arnettj_rest file.

Open http://localhost:5173/ in a browser.
Create, edit, or delete exercises within the online exercise collection shown, and save your data on the MongoDB cloud to access again later!
